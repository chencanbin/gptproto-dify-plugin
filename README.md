# GPTProto Tools Plugin

AI tools powered by GPTProto API.

## Features

- **Gemini Text to Image**: Generate high-quality images from text prompts using Google Gemini model

## Installation

1. Download the `.difypkg` file
2. Go to Dify Plugin Management
3. Upload and install the plugin
4. Configure your GPTProto API key

## Get API Key

Visit [GPTProto](https://gptproto.com/apis/list) to get your API key.

## Tools

### Gemini Text to Image

Generate images from detailed text descriptions.

**Parameters:**
- `prompt` (required): Text description of the image
- `size`: Image resolution (1K or 2K)
- `aspect_ratio`: Image aspect ratio (1:1, 16:9, 9:16, 4:3, 3:4)
- `output_format`: Output format (png or jpeg)
